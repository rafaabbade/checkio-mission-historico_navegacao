Como armazenar as páginas visitadas?

Usaremos duas pilhas (historico e futuras) para armazenar as páginas visitadas e para gerenciar a navegação de "voltar" e "avançar". A pilha historico armazena as páginas visitadas no passado, enquanto a pilha futuras armazena as páginas que foram visitadas antes do comando "voltar", permitindo retornar a essas páginas com o comando "avançar".

Como implementar o comando "voltar"?

Para implementar o comando "voltar", você precisa desempilhar a URL do topo da pilha de navegação atual e empilhar na pilha de páginas futuras (futuras). Exemplo: 

futuras = []
if historico:
    futuras.append(atual)
    atual = historico.pop()

Como implementar o comando "avançar"?

Para implementar o comando "avançar", você precisa desempilhar a URL do topo da pilha de páginas futuras e empilhar novamente na pilha de navegação atual. Exemplo: 

if futuras:
    historico.append(atual)
    atual = futuras.pop()

Como iniciar a navegação?

Inicie a navegação com a URL atual definida como "Nenhuma página visitada" e as pilhas (historico e futuras) vazias. Quando um comando de visita for executado, atualize a URL atual e limpe a pilha de páginas futuras.

Como entender e separar os comandos?

Para entender e separar os comandos, você pode usar a função split() em cada comando. Por exemplo, para separar o comando "visitar www.google.com" e extrair a URL, use o seguinte código: 

comando = "visitar www.google.com"
partes = comando.split()
acao = partes[0]  # 'visitar'
url = partes[1]  # 'www.google.com'
